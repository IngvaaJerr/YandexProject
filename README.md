Автотест поиска заказа по track

data.py: order_body - тело запроса создания заказа, headers - заголовки запроса
configuration.py - составные части URL для запросов
post_and_get_requests.py - post запрос на создание заказа и get запрос на поиск заказа по его track номеру
test_diplom.py - автотест проверки статуса запроса заказа по его track номеру на соответствие статусу 200

Всё вместе - Создание заказа с телом запроса, запрос информации о заказе через его track номер и автотест проверки статуса запроса на соответствие значению 200
